Qu'est-ce que le JavaScript ?

Le langage de base du web

INTRODUCTION

L'analogie de la maison

Poursuivons l'analogie. Si HTML est la structure et CSS l'apparence, JavaScript est le système électrique et les automatismes de la maison. Il permet d'allumer les lumières, d'ouvrir les portes, de déclencher des alarmes, etc., en réponse aux actions des occupants.

Définition claire et concise

JavaScript est un langage de programmation de haut niveau, interprété, qui permet d'ajouter des fonctionnalités interactives à des pages web. Il s'exécute directement dans le navigateur de l'utilisateur.

Importance fondamentale

JavaScript est devenu indispensable pour créer des expériences utilisateur modernes et dynamiques, allant des simples animations aux applications web complexes. vous serez en mesure de développer des sites web dynamiques et interactifs.

LES BASE DU JAVASCRIPT

Les variables et fonctions

Une variable est un conteneur qui stocke une valeur (nombre, texte, booléen, etc.)

Une fonction est un bloc de code réutilisable qui effectue une tâche spécifique.function saluer(nom) { console.log("Bonjour " + nom + " !"); }

Les structures de contrôle

Les structures conditionnelles (if, else) permettent d'exécuter du code en fonction d'une condition.

Les boucles:(for, while)permettent de répéter une instruction plusieurs fois.

Le DOM

Le DOM est une représentation de la page HTML sous forme d'arbre d'objets.

Il permet de manipuler le contenu, la structure et le style des éléments HTML

Exemple : document.getElementById("monElement").style.color = "blue";

CODE JAVASCRIPT

Comprendre le JAVASCRIPT

Le JavaScript est le langage de base pour rendre les pages web interactives. Il permet de manipuler le contenu, de réagir aux actions des utilisateurs, et d'ajouter des fonctionnalités dynamiques. Dans cette section, vous pouvez voir un exemple de code en JAVASCRIPT 

        const boxes = document.querySelectorAll('.item');
        let animationActive = false;
        
        function toggleAnimation() {
            animationActive = !animationActive;
            boxes.forEach((box, index) => {
                if (animationActive) {
                    setTimeout(() => {
                        box.style.transform = 'scale(1.5) rotate(45deg)';
                    }, index * 500);
                } else {
                    box.style.transform = 'scale(1) rotate(0deg)';
                }
            });
        }
