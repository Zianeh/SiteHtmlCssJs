Qu'est-ce que le HTML ?

Le langage de base du web

INTRODUCTION

L'analogie de la maison

Poursuivons cette analogie. Si le HTML est le plan de la maison, les briques seraient le contenu (texte, images, etc.) que l'on vient placer dans cette structure.

Définition claire et concise

HTML est un langage de balisage qui sert à structurer le contenu d'une page web. Chaque balise indique au navigateur comment afficher un élément spécifique (un titre, un paragraphe, etc.).

Importance fondamentale

Sans HTML, il serait impossible de créer des pages web telles que nous les connaissons. C'est le socle sur lequel repose tout le développement web.

LES BASE DU HTML

Les balises

Une balise est délimitée par des chevrons (< et >). La plupart des balises ont une balise de fermeture.

Exemple : <p>Ceci est un paragraphe.</p>

La structure d'un document HTML

<!DOCTYPE html>: Indique au navigateur qu'il s'agit d'un document HTML5.

<html>: Racine de tout document HTML.

<head>: Contient des informations sur la page.

<body>: Contient le contenu visible de la page.

Les éléments les plus utilisés

Paragraphes : <p></p>

Titres : <h1> à <h6>

Listes : <ul> ou <ol> avec <li>

Liens : <a href="https://www.example.com">

Images : <img src="image.jpg" alt="Description">

CODE HTML

Comprendre l'HTML

Le HTML est le langage de base pour créer des pages web. Il permet de structurer le contenu avec des balises. Dans cette section, vous pouvez voir un exemple de code en HTML.
