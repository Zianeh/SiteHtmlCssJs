Qu'est-ce que le CSS ?

Le langage de base du web

INTRODUCTION

L'analogie de la maison

Poursuivons l'analogie. Si le HTML est le plan de la maison, le CSS est la décoration intérieure et extérieure. Il détermine la couleur des murs, le choix du mobilier, l'agencement des pièces, etc.

Définition claire et concise

CSS est un langage de style utilisé pour décrire la présentation des documents HTML. Il permet de contrôler l'apparence visuelle des éléments d'une page web.

Importance fondamentale

Le CSS est essentiel pour transformer un ensemble de balises HTML en une interface utilisateur agréable et cohérente. Il permet de créer des sites web esthétiques et adaptés à tous les écrans.

LES BASE DU CSS

Les sélecteurs

Un sélecteur permet de cibler les éléments HTML auxquels on veut appliquer un style.

-p: Tous les paragraphes

-.classe: Tous les éléments ayant la classe "classe"

-#id: L'élément ayant l'ID "id"

Les propriétés et les valeurs

-Les propriétés CSS définissent les caractéristiques d'un élément color, font-size

-Les valeurs indiquent la valeur à attribuer à une propriété.color: blue: font-size: 100px

URLs: Pour les propriétés comme: background-image

Les feuilles de style

-Un fichier CSS est un fichier texte avec l'extension: .css

-Il contient l'ensemble des règles CSS d'unsite

-Il est lié à un fichier HTML via la balise<link>

CODE CSS

Comprendre le CSS

Le CSS est le langage de base pour styliser des pages web. Il permet de définir l'apparence des éléments avec des sélecteurs et des propriétés. Dans cette section, vous pouvez voir un exemple de code en CSS 

.container {
display: flex;
justify-content: space-around;
align-items: center;
height: 100vh;
}

.item {
background-color: lightblue;
padding: 20px;
margin: 10px;
border-radius: 5px;
}
