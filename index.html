Quel langage de programmation choisir ?

HTML (HyperText Markup Language) est le langage standard utilisé pour créer les pages web que vous consultez tous les jours. C'est un peu comme la grammaire du web : il définit la structure et le contenu d'une page.
CSS (Cascading Style Sheets) est un langage de style utilisé pour décrire la présentation des documents HTML et XML. En d'autres termes, si HTML est le squelette d'une page web, CSS en est l'habillage.
JavaScript est un langage de programmation de scripts principalement employé dans les pages web interactives. Il est souvent considéré comme le troisième pilier du développement web, aux côtés d'HTML et de CSS
